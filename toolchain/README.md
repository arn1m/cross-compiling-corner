# Compiler Toolchains

## Links
- newest ARM toolchains: https://developer.arm.com/downloads/-/arm-gnu-toolchain-downloads
- older ARM toolchains for Cortex-A: https://developer.arm.com/downloads/-/gnu-a
- older ARM toolchains (bare-metal only) for Cortex-A/M/R: https://developer.arm.com/downloads/-/gnu-rm

## Notes
For embedded Linux targets, it is advisable to use the same (or very similar) cross-compiler version than the compiler installed with the embedded Linux.
